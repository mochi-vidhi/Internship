# Student Information Portal

This is a simple web application built with **React** and **Appwrite** to manage student details like name, age, field, and grade.

## 🚀 Features

- Add new student details
- View a list of all students
- Connected to Appwrite backend for database operations

## 🛠️ Tech Stack

- React JS
- Appwrite (Database)
- HTML & CSS
